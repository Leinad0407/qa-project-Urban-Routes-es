# Urban Routes

## Urban Routes es una aplicación para pedir transporte, similar a Uber o Didi. Dicha aplicación te permite seleccionar entre varios tipos de transporte y sus diferentes modos, los cuales, dependiendo de la selección del usuario, te permite agregar topings en la renta del vehículo.

## El alcance de este proyecto solo cubre pruebas generales en la página principal.

### Como iniciar el proyecto:
- Necesitas tener instalados los paquetes pytest y selenium para ejecutar las pruebas.
- Agrega la URL del servidor en el archivo configuration.py.
- Ejecuta todas las pruebas ubicadas en el archivo main con el comando pytest.
